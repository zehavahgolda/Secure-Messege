7 Python-Specific Questions

Why did you use @dataclass? What does frozen=True mean?
  • @dataclass - automatically creates __init__, __repr__, __eq__
  • frozen=True - makes the object immutable (fields cannot be changed)
@dataclass(frozen=True)
class TimeRange:
    start: time
    end: time

# This will raise an error:
tr = TimeRange(time(9,0), time(10,0))
tr.start = time(8,0)  # FrozenInstanceError!
Do you use List Comprehensions or explicit loops?
List Comprehension - good for simple mapping/filtering 
free_slots = [slot for slot in slots if slot.duration >= min_duration]
Explicit loop - good for complex logic 
merged = []
for event in sorted_events:
    if merged and event.start <= merged[-1].end:
        merged[-1] = merge(merged[-1], event)
    else:
        merged.append(event)
Show me how you parse time strings. Do you handle format errors?
from datetime import datetime

def parse_time(time_str: str) -> time:
    """Parse time string in HH:MM format."""
    try:
        return datetime.strptime(time_str.strip(), "%H:%M").time()
    except ValueError as e:
        raise InvalidTimeFormatError(
            f"Invalid time format: '{time_str}'. Expected HH:MM"
        ) from e
